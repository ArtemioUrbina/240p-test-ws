# 240p-test-ws (144p Test Suite for WS)

A work-in-progress implementation of [Artemio's 240p Test Suite](https://junkerhq.net/xrgb/index.php?title=240p_test_suite)
for the WonderSwan.

Currently implemented:

- PLUGE (WS/WSC)
- Color bars (WSC only)
- Grid (WS/WSC)
- Drop shadow test (WS/WSC)
